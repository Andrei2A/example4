# Калькулятор

Веб-приложение калькулятора с современным дизайном, написанное на чистом HTML, CSS и JavaScript.

## Описание

Это полнофункциональный калькулятор с интуитивным интерфейсом в тёмной теме. Приложение работает полностью на стороне клиента и не требует сервера или дополнительных зависимостей.

## Функционал

### Математические операции
- **Сложение** (+) — сложение двух чисел
- **Вычитание** (−) — вычитание одного числа из другого
- **Умножение** (×) — умножение двух чисел
- **Деление** (÷) — деление одного числа на другое (с защитой от деления на ноль)
- **Проценты** (%) — преобразование числа в проценты (деление на 100)

### Дополнительные функции
- **C** — полная очистка калькулятора (сброс всех значений)
- **DEL** — удаление последнего введённого символа
- **Десятичная точка** (.) — ввод дробных чисел

### Поддержка клавиатуры
| Клавиша | Действие |
|---------|----------|
| 0-9 | Ввод цифр |
| + - * / | Математические операции |
| Enter или = | Вычислить результат |
| Escape | Очистить калькулятор |
| Backspace | Удалить последний символ |
| % | Проценты |
| . | Десятичная точка |

## Как запустить

### Способ 1: Открыть напрямую
1. Перейдите в папку проекта
2. Дважды кликните на файл `index.html`
3. Калькулятор откроется в браузере по умолчанию

### Способ 2: Через командную строку (Windows)
```cmd
cd C:\develop\example4
start index.html
```

### Способ 3: Через PowerShell
```powershell
Invoke-Item "C:\develop\example4\index.html"
```

### Способ 4: Локальный сервер (опционально)
Если у вас установлен Node.js:
```cmd
npx serve .
```
Или Python:
```cmd
python -m http.server 8000
```
Затем откройте в браузере: `http://localhost:8000`

## Технические требования

- Любой современный браузер (Chrome, Firefox, Edge, Safari)
- JavaScript должен быть включён в браузере

## Особенности реализации

- **Без зависимостей** — не требует библиотек или фреймворков
- **Адаптивный дизайн** — корректно отображается на разных устройствах
- **ООП-архитектура** — логика реализована через класс Calculator
- **Обработка ошибок** — защита от деления на ноль
- **Цепочка операций** — поддержка последовательных вычислений
